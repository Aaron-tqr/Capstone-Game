# EMOTICONS: A Mobile Game for Teaching Emotion Recognition and Vocabulary to Children Ages 3-5

EMOTICONS is a capstone game built with Godot for mobile devices. It is designed to help children ages 3-5 learn to recognize emotions, connect emotions to situations, and build early vocabulary through colorful characters, interactive scenes, and guided play.

## Overview

The game introduces five core emotions:

- Joy
- Sadness
- Anger
- Disgust
- Fear

Players progress through story scenes and matching challenges that reinforce emotional understanding through visuals, voiceover, and simple touch interaction.

## Key Features

- Story-based learning that presents emotions in context
- Emotion Matching mode for identifying the correct emotion from a situation
- Word Matching mode for building emotion vocabulary through drag-and-drop play
- Unlockable content that encourages progression
- Heart and star systems that support replayability and feedback
- Mobile-friendly interface designed for touch input
- Audio, animation, and visual cues to support young learners

## Game Modes

### Story Mode

Story Mode introduces the emotions through interactive scenes and dialogue. Players respond to questions during the story by choosing the emotion that best matches the situation.

Progress through Story Mode unlocks additional content in the game.

### Emotion Matching

In Emotion Matching, players are shown a scenario and must tap the emotion that best fits the situation. Correct answers earn stars, while incorrect answers reduce hearts.

This mode is designed to strengthen emotion recognition and situational reasoning.

### Word Matching

Word Matching focuses on vocabulary development. Players match emotion words and related content through drag-and-drop interactions.

This mode builds on the emotion recognition skills developed in the earlier game modes.

## How To Play

- Read or listen to the prompt shown on the screen
- Observe the situation or character expression
- Select the emotion or word that best matches the prompt
- Earn stars for correct answers and complete levels to unlock more content
- Retry levels to improve your score and understanding

## Controls

The game is designed for touch devices, but it can also be played with a mouse in the Godot editor.

- Tap buttons to navigate menus and select answers
- Drag items in Word Matching levels
- Use the pause menu to resume, retry, access settings, or return to the main menu

## Requirements

- Godot 4.6 or newer
- An Android device or emulator if you want to test the mobile build

## Running The Project

### In Godot

1. Open the project in Godot.
2. Run the main scene from the editor.
3. Use the menus to start Story Mode, Emotion Matching, or Word Matching.

### Android Build

The project includes Android export files for building a mobile APK.

1. Open the project in Godot.
2. Configure your Android export settings.
3. Export the project as an APK for testing on a mobile device.

## Project Structure

- `scenes/` - Main game scenes and UI scenes
- `scripts/` - Gameplay logic, scene transitions, audio, and state management
- `assets/` - Visual and audio assets used in the game
- `android/` - Android build support files
- `export/` - Export artifacts and generated files

## Educational Goal

This capstone project is intended to support early childhood learning by helping young children:

- Recognize common emotions
- Associate emotions with situations
- Learn emotion-related vocabulary
- Practice memory, attention, and decision-making through play
