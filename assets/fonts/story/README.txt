Story fonts (recommended for kids 3–5)
=====================================

1. Download from Google Fonts (free):
   - Fredoka (Bold for titles & Next button; Regular for dialogue)
   - Nunito (optional, for dialogue body)

2. Put the .ttf files in THIS folder:
   res://assets/fonts/story/

   Suggested filenames (after you download):
   - Fredoka-Bold.ttf
   - Fredoka-Regular.ttf
   - Nunito-Regular.ttf (optional)

3. In Godot: select the story scene → DialoguePanel / labels / buttons →
   Inspector → Theme Overrides → Fonts → Font → load your .ttf

4. Re-run the game. (You can also create a Theme resource once and reuse.)

Tip: If you skip custom fonts, Godot will use the default project font — layout still works.
